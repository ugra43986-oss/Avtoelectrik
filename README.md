# Автосервис в п. Угра

Готовый архив структуры сайта под GitHub Pages.

## Что внутри
- `index.html` — основной файл сайта
- `images/hero-service.jpg` — главное фото первого экрана
- `images/gallery-1.jpg` — главное фото галереи
- `images/gallery-2.jpg` — фото подъемника
- `images/gallery-3.jpg` — фото диагностики
- `images/gallery-4.jpg` — фото ремонта / автоэлектрики

## Что нужно сделать
1. Заменить картинки в папке `images` на свои реальные фото
2. При необходимости поправить цены в `index.html`
3. Загрузить содержимое репозитория на GitHub
4. Включить GitHub Pages:
   - Settings
   - Pages
   - Deploy from a branch
   - Branch: `main`
   - Folder: `/ (root)`
